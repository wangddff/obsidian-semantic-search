## 🛠️ 部署与使用

### 快速开始
```bash
# 克隆项目
git clone https://github.com/yourusername/obsidian-semantic-search.git
cd obsidian-semantic-search

# 安装依赖
pip install -r requirements.txt

# 测试系统
python cli.py test

# 处理笔记库
python cli.py process /path/to/obsidian/vault

# 搜索内容
python cli.py search "人工智能是什么" --limit 5
```

### Docker部署
```bash
# 构建镜像
docker build -t obsidian-semantic-search .

# 运行容器
docker run -v ./data:/app/data -v ./config:/app/config obsidian-semantic-search

# 使用Docker Compose
docker-compose up -d
```

### 配置说明
- **LM Studio API**: 需要本地运行LM Studio并加载BGE-M3模型
- **路径配置**: 支持自定义数据目录和配置文件路径
